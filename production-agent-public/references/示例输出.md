## 标准输出示例

任务是定时抓取网页 + 本地持久化。生产隐患扫描：

```python
# config.py
TARGET_URL   = "https://example.com/data"
INTERVAL_MIN = 30        # 每30分钟抓取一次
DB_PATH      = "./data.db"
MAX_RETRIES  = 3

```

```python
# fetcher.py
import requests, time, logging
from config import TARGET_URL, MAX_RETRIES, REQUEST_TIMEOUT

logger = logging.getLogger(__name__)

HEADERS = {"User-Agent": "Mozilla/5.0 ..."}

def fetch(url: str = TARGET_URL) -> str | None:
    for attempt in range(1, MAX_RETRIES + 1):
        try:
            resp = requests.get(url, headers=HEADERS, timeout=REQUEST_TIMEOUT)
            resp.raise_for_status()
            return resp.text
        except Exception as e:
            wait = 2 ** attempt
            logger.warning("第%d次失败: %s，%d秒后重试", attempt, e, wait)
            if attempt < MAX_RETRIES:
                time.sleep(wait)
    logger.error("已达最大重试次数，本次抓取放弃")
    return None
```

**Observation 2：**
指数退避：1次失败等2s，2次等4s，3次等8s，符合生产标准。

